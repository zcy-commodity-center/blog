Alfred Wrokflow - Zcy Jenkins

2019-05-16 效能提升 Alfred Workflow, Jenkins 0 评论 字数统计: 409(字) 阅读时长: 1(分)
Key Value
Jenkins构建项目、更新项目、快速跳转
快捷键 jc、ju、jj
下载 click me

配置

在用户根目录~/下创建文件.zcy_alfred，配置如下内容（注意标点符号需要为英文）并保存

开发同学账号没有权限更新测试环境项目。若有需要，在允许的情况下，可向测试同学索要账号并配置如下内容

其中jenkins token获取步骤如下：

jenkins -> 个人中心（右上角） -> Configure -> Add new Token -> 输入名称点击Generate即可

注：开发同学若配置了测试账号，只有在更新测试环境的项目时才会使用测试同学的账号

使用

构建项目 - jc

  1. 唤起alfred
  2. 输入应用名关键字（模糊匹配，回车选择）
  3. 输入分支关键字（模糊匹配，回车选择）
  4. 根据应用和分支进行编译（回车进入任务页面）

更新项目 - ju

  1. 唤起alfred
  2. 输入应用名关键字（模糊匹配，回车选择）
  3. 选择构建好的包（回车选择）
  4. 输入环境关键字（模糊匹配，回车选择）
  5. 根据应用、包和关键字进行更新（回车进入任务页面）

快速跳转 - jj

  1. 唤起alfred
  2. 输入环境关键字（模糊匹配，回车选择）
  3. 回车进入对应环境的视图页面
